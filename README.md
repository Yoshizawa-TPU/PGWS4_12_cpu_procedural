### あなたの考えた素敵なシェーダー

![結果](https://github.com/user-attachments/assets/1e3afb9e-1646-46bc-8187-a076f354bba2)


- 工夫した点
  - GenerateのHEIGHTの値を500F~1000Fで取ることで、雷の長さが不均一になっています。
  - オブジェクトの生成を3～10のランダムの数で生成しています。
  - 以上の事により、ランダムな雷が意味のあるパターンのように見えるパレイドリアができました
  - 1枚目タンゴを踊る女性、二枚目は親子のように見えました。
